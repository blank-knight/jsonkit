# SEO 副业 P0 规划

> 起点：参考百年（@yidabuilds）4 个月 10 万+ 的副业打法，结合自身情况裁剪。
> 目标：跑通"SEO 站 + 外链系统 + Skill 包"这条复利资产线。

---

## 一、百年打法的复盘

### 收入结构（4 个月）
- 企业定制项目：7w+（抖音图文获客 → 1 个匹配大单）
- 教培收入：3w（旧业务存量）
- Skill 包/方法论：几千
- SEO 广告：月几十美金，增长中

### 真正可复用的方法论
- Claude Code 工作流：Skill / Agent / Hook / 凭证管理
- Astro + Tailwind + Cloudflare Pages 的建站流水线
- 竞品外链反查 + Playwright MCP 半自动提交
- 把自用流程打包成产品卖给社群

### 不可复现的部分（运气/存量）
- 抖音 2026 年初图文红利窗口
- NUS + 跨境电商主业的身份钩子
- 4~5 年留学辅导的存量收入安全垫
- 7~8 个客资里筛出 1 个匹配项目的运气

### 核心心法
> **代码 AI 能写，真正值钱的是流程数据。**
> 每次干完要留下可复用/可售的数据资产，不是一次性交付。

## 二、对小红书 AI 图文号的判断

可以做，但**不放在 P0**。理由：
- "窗口期" 是营销话术，3~6 个月内不会消失
- 起号 ≠ 变现（百年抖音 9 篇 3 篇 10w+ 最后还是停更）
- 先把变现端搭好再开流量水龙头，否则一堆粉不知道往哪导
- 真要做就做 "AI 副业 / 独立开发" 垂直细分，不做泛知识号

## 六、关键策略原则

### 1. 选词避开 AI Overview 死区
2026 年 Google AI Overview 已吃掉大量信息查询点击。要做：
- **工具型查询**（converter / generator / calculator）—— AI 答不了，必须点站
- **本地化查询**（带地区/语言/法规）
- **登录后才能用的工具站** —— AdSense + 订阅双轨

### 2. 优先做英文站
中文 SEO 流量便宜但 AdSense 单价低 10 倍，工程量一样，回报差距巨大。

### 3. 外链流程库从 day 1 当产品做
- Schema 一开始就按"可售卖"设计：成功率 / 更新时间 / 地区 / 失效告警
- 自己用的同时跑 5~10 个种子用户
- 真正护城河是**持续维护的平台失效告警**

### 4. 不烧双 Max Plan
百年能撑是因为有 7w 大单 + 3w 教培托底。先跑通一个赛道再加杠杆。

## 七、今天要定的 3 件事（待用户决定）

### 1. 赛道方向
候选（按开发者背景适配度排序）：

- **A. 开发者工具站**（AI 提示词生成器 / 代码格式化 / 正则测试 / JSON 工具）
  - 优势：最熟，AdSense + 联盟双轨
  - 劣势：竞争最激烈，DR 起步难

- **B. AI 工具垂直比对站**（某类 AI 工具的横评/导航）
  - 优势：长尾词多，AI 工具迭代快带来增量
  - 劣势：内容更新维护成本高

- **C. 出海 SaaS / 独立开发资源站**（英文，给海外开发者）
  - 优势：AdSense 单价最高
  - 劣势：英文内容质量门槛高

### 2. 预算

**采用阶梯方案，不一开始就上 Ahrefs。**

#### 阶段 A：验证期（第 1~4 周）≈ $1/月
- **AI**：单 Max Plan（已有）；外链自动化的长任务走 API 按量计费
- **关键词**：GSC + Ahrefs Webmaster Tools（自家域免费）+ Google Keyword Planner + Claude 跑长尾扩展
- **竞品反查**：Ahrefs Free Backlink Checker（每域 100 条）+ openlinkprofiler.org
- **域名**：Cloudflare Registrar（.com 成本价 $9~10/年，和 Pages 一体化；不要 GoDaddy，续费会涨）
- **部署 / DB**：Cloudflare Pages + 本地 Docker PG，全免费

#### 阶段 B：第 1 个站有流量信号后，突击 1 个月 Ahrefs Starter（$129 × 1）
- 集中一周跑完：所有竞品反查 + 5 个站的选词 + 外链平台清单
- 数据导出到本地，**退订**
- Ahrefs 是"集中挖一次就能拿走 80% 价值"的工具，没必要月月续

#### 阶段 C：长期续订门槛
- 单月 AdSense 收入 > $200 才续 Ahrefs
- 单月主动收入 > 双 Max 成本的 5 倍才考虑加第二个 Max
- 否则杠杆都按"亏损项"处理，回阶段 A

### 3. 每天投入时长
决定 4 周节奏还是 8 周节奏
